# 🐦 Water-redstart: the chirping viewer v2.8 (双平台版) - 极速鸟类照片挑选小助手

## 🪟 Windows 使用说明
本程序为封装好的单文件应用程序，**无需安装任何环境，直接将 `Water-redstart_v2.8.exe` 复制或移动到你的照片文件夹内，双击运行即可开始挑图！**（如果遇到U盘/SD卡中弹窗报错，你也可以把它放在桌面，然后点击左上角 `文件` -> `打开照片文件夹...` ，手动选择你想看图的文件夹即可！）

---

## 🍎 macOS 小白一键运行指南

由于本程序现已全面支持 Windows / macOS 双平台运行，对于不熟悉终端操作的 macOS 电脑小白用户，您可以通过以下方式轻松制作一个“双击即用”的脚本：

1. 在本程序所在文件夹中，新建一个纯文本文件，命名为 `启动看鸟.command`（注意后缀名是 `.command`）。
2. 使用普通的文本编辑器（如 Mac 自带的文本编辑）将其打开，将以下代码完整复制并粘贴进去，然后保存：

```bash
#!/bin/bash
cd "$(dirname "$0")"
pip3 install pillow rawpy
python3 photo_viewer.py
```

3. **首次权限激活（仅需一次）**：
   打开 Mac 的“终端（Terminal）”，输入 `chmod +x `（注意 `+x` 后面必须保留一个空格），然后直接将刚才创建的 `启动看鸟.command` 文件用鼠标拖到终端窗口里，按下回车即可。

完成以上步骤后，以后每次在 Mac 上看图只需 **双击 `启动看鸟.command`** 就能直接打开软件啦！

## ✨ 软件简介
Water-redstart 是专为摄影师、观鸟记录者以及需要从海量照片中**快速筛选出图**的用户打造的一款**本地极速看图工具**。

告别缓慢的文件夹拖拽和繁琐的复制粘贴操作，通过 Water-redstart: the chirping viewer，你只需使用键盘热键进行“快准狠”的二元判断（跳过或挑出），即可用极高效率梳理成百上千张单反照片。它被设计得轻量、活泼且极度流畅。

## 🚀 核心功能亮点

### 1. 极致交互：全键盘沉浸式筛选
- **[←] 左键**：回到上一张照片。
- **[→] 右键 / [D]键**：这张一般般，跳过下一张！
- **[↑] 上键 / [W]键**：这张很棒！一键选出并触发清脆的“叽叽”鸟鸣音效，自动将当前源文件安全复制到选中的独立子文件夹，毫不拖泥带水。
- **[↓] 下键 / [S]键**：看到一张想分享的高光时刻？直接将源文件路径放入系统剪切板（彻底解决大分辨率照片复制后提及膨胀变数百兆的问题），随时完美无损地粘贴到微信或文档中。
- **[A] 自动撤销**：手滑了？别着急，按下撤销键，自动退回上一张。如果是“挑出”手滑，程序会自动跟进帮你把刚才放进子文件夹里的废片删干净（放飞小鸟）。
- **顶端灵动进度条**：上方自带可视化进度条，使用鼠标拖动它就可以在全盘大跨度照片中进行快进和翻页！
- **灵活的倒序审图**：开启设置里的“倒序选片”，系统将一秒自动头尾翻转，方便您顺着刚刚拍摄完成的最新照片一直往前看！
- *以上所有热键均可随时进入设置面板修改为您习惯的顺手按键，哪怕是方向键也同样支持哟！*

### 2. ⚡ RAW/巨型原图极速翻页体验
告别每翻一张 30MB 的原图就要卡顿转圈的噩梦！
- **全面支持更多 RAW 格式**：本版本新增支持 `.naw` 与 `.nrw` 格式文件，满足更多型号相机的看图需求。
- **重构的底层极速通道**：通过拦截并使用底层 C 库在图片读取阶段完成分辨率的降采样解码，使得巨型文件占用内存减半且画质锐利不降，体验飞升。
- **后台多线程深度预读**：它像个聪明的小管家，在你看眼下这张时，它在后台已将后多达**五十张**照片悄悄放入内存缓存。每一次按键翻页都是“秒切”，哪怕以“一拍三张”的节奏盲开连按，也绝不会有一丝卡顿。

### 3. 🖱️ 沉浸式拖拽放大
- **鼠标滚轮局部缩放**：看鸟当然要数毛！使用鼠标滚轮可随心所欲悬停放大照片局部的细节。
- **左键平滑拖拽**：放大后支持鼠标左键按住拖动自由平移，享受顺滑细节审视体验！

### 4. 📂 智控文件夹与善后清理
- **智能记忆读图进度**：不想下次从大分类的第一张重看？不慌！软件带有每次随用随保存的功能，下一次打开它会在上次结束的地方等着您。
- **贴心多功能结尾向导**：当你梳理完该目录下最后一张照片，程序会立刻为你颁发“小红花”，并直接弹出快捷处理面板：你可以选择一键无痕抹除已经挑废了的整个文件夹原片，甚至还可以让老版本看图程序上演“大义灭亲”——顺手把自己也删除得干干净净。
- **省内存模式 (v2.3新增)**：在设置菜单中开启时，预读数量自动从50张下降到20张，特别适合配置压力较大的老电脑或巨型RAW照片库的审图场景。

## 🔧 [v2.8] 更新日志
1. **新增**：加入了当无可显示的图片时的引导欢迎页，支持原生拖拽文件夹直接开启看图选图（现已加入 windnd 支持）。
2. **新增**：支持同名多格式文件（如 JPG+RAW）重叠显示与处理功能。优先展示 JPG 以加速预览，挑出时自动一并打包所有同名格式文件，复制剪贴板时则仅复制 JPG；且状态栏会有相应重叠显示提示，一切尽在后台快速完成。

## 🔧 [v2.7] 更新日志
1. **新增**：补全并全面囊括了各家相机厂商的 RAW 格式支持！除原有的格式外，最新版本现在完美支持索尼（`.arw`, `.sr2`, `.srf`）、佳能（`.crw`, `.cr2`, `.cr3`）、尼康（`.nef`, `.nrw`）、松下（`.rw2`）、奥林巴斯（`.orf`），以及富士（`.raf`）、宾得（`.pef`）和通用（`.dng`）格式照片！
2. **新增**：支持完全自定义挑出照片的目标文件夹！在“设置”菜单修改文件夹名的后方加入了“浏览”按钮，您可以自由选择电脑中的任意硬盘或路径作为集中收件箱。且该设定会自动随当前看图文件夹存档，换个文件夹重看仍然默认使用原来的“SELECT”。
3. **优化**：大幅提升了按下“挑选（W 键）”时的软件响应速度！现在选片复制文件的动作已完全放入后台多线程处理，对于巨大体积的 RAW 图片，按下快捷键即刻秒切下一张，再也不用卡顿等待文件复制完成了。
4. **优化**：在选完照片后若选择“删除本程序”，如果检测到当前程序未存放在该照片文件夹下，将会保留本程序，仅擦除该文件夹下的记忆配置文件，避免误删程序。

## 🔧 [v2.6] 更新日志
1. **新增**：新增键盘上下左右键的控制方式，可代替键盘的这四个默认控制键。下键能够快速分享照片的物理路径，这些方向功能也可以同其他几个热键槽位完全独立，在主界面的“修改操作热键与文件夹”控制台中自行调配绑定与热键更替！

## 🔧 [v2.5] 更新日志
1. **新增**：增加了照片显示画质调节功能（在“设置”菜单中），默认升级为极具震撼力的 **8K 无损级别** 检视体验，让“数毛”更加纤毫毕现。
2. **优化**：提供了 2K（运行快、省内存）、4K（平衡）选项以适应老旧电脑，且您的画质选择会自动保存，下次直接生效。

## 🔧 [v2.4] 更新日志
1. **新增**：支持菜单栏“文件 - 打开照片文件夹...”，完美解决直接在U盘或外接存储卡中运行程序时可能爆发的内存读取权限报错。
2. **新增**：启动自动切换英文输入法功能，针对 Windows 系统，打开软件将自动向系统申请激活美式英语键盘模式，彻底根治中文输入法拦截选图快捷键的尴尬。
3. **优化**：画质呈现引擎大幅升级！针对鼠标滚轮无级放大的局部深视场景，程序底层的分辨率缓存阈值由 3000px 提标扩容至惊人的 6000px，并且增加了高阶双三次滤波与 Lanczos 锐利重构抗锯齿计算。现在，尽情拉满滚轮享受最丝滑的“数毛”吧！
4. **新增**：鸣谢名单加入白鹡鸰、灰喜鹊、碳酸。

## 🔧 [v2.3] 更新日志
1. **新增**：在设置菜单中增加“全屏时保留上方菜单栏”开关，全屏沉浸看图还是保留进度信息可由您随心分配，且支持永久记忆。
2. **新增**：菜单内增加“省内存模式”选项，在小内存设备也可丝滑挑图。
3. **修复**：最后一张图多次点击“下一张”热键会导致弹窗重复弹出的逻辑错误。
4. **新增**：鸣谢名单加入瑞瑞的。
